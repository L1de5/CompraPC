# CompraPC

### Trabalho de conclusão da disciplina Desenvolvimento de software II.

O trabalho consiste em criar um site de venda de computadores utilizando a linguagem de programação Python com o auxílio do [Flask](http://flask.palletsprojects.com/en/1.1.x/). O projeto visa agrupar uma série de técnologias apresentadas durante o 3º Bimestre, como por exemplo: Flask-Login, SQLAlchemy e WTForms.
